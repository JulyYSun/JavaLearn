# 随机数
1. Math.random()返回一个0~1的double随机数，获取更大的随机数可以通过运算操作放大实现
2. java.util.Random类  
```
Random rnd=new Random();  
int rndint=rnd.nextInt(n);  
```
返回一个0~n（不包括n）的随机整数，更多随机数生成函数参考文档
